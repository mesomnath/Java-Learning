# Java Basics - For Loops Examples

Simple Java examples demonstrating basic for-loop usage and patterns.

## Overview

This repository contains small, self-contained Java programs that illustrate for-loops. Example: `src/javaBasics/forLoops/StarPattern.java` prints a right-angled star pattern.

## Requirements

- Java JDK 8 or newer
- IntelliJ IDEA (optional) or any Java-capable IDE
- Command-line tools: `javac`, `java`

## Build & Run (command line - Windows)

1. Compile: javac -d out src\javaBasics\forLoops\StarPattern.java
2. Run: java -cp out javaBasics.forLoops.StarPattern


## Run in IntelliJ IDEA

- Open the project folder in IntelliJ IDEA.
- Mark `src` as Sources Root if required.
- Run the `StarPattern` class from the editor or create a Run Configuration with main class `javaBasics.forLoops.StarPattern`.
